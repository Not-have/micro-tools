@mt-kit/conf ​

配置文件集合，提供常用的配置常量和枚举值。

安装 ​

bash
npm i @mt-kit/conf -D

API ​

阿里矢量图标库 ​

ICONFONT_URL ​

阿里矢量图标库（iconfont）的在线地址常量。

ts
import { ICONFONT_URL } from '@mt-kit/conf';

// 在 HTML 中使用
<link rel="stylesheet" href={ICONFONT_URL} />

IconType ​

图标类型枚举，用于类型安全的图标使用。

ts
import { IconType } from '@mt-kit/conf';

// 使用示例
const icon: IconType = IconType.SUCCESS;
